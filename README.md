# CSRF Protection in Express.js using Cookies

This project demonstrates how to implement **CSRF (Cross-Site Request Forgery) protection** in an Express.js application using the `csurf` middleware and cookies.  
It includes a simple form rendered with **EJS** and securely validates CSRF tokens on form submission.

## Features
- CSRF protection using `csurf`
- Cookie-based token storage
- Secure form handling with EJS
- Simple and beginner-friendly setup

## Technologies Used
- Node.js
- Express.js
- csurf
- cookie-parser
- EJS
